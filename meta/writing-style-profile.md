# Writing Style Profile

> The Editor uses this file to track the writer's voice and style over time.
> Updated after review sessions as patterns emerge.

---

## Voice & Tone
Dry, deadpan, lightly ironic. The narrator observes both the game world and the real person behind the screen with the same detached clarity — almost documentary in its cool remove. The humor never announces itself; it arrives through juxtaposition and understatement. The tone never condescends to its subject, which gives the irony real warmth underneath.

## Sentence Style
Confident mix of short punches and longer informational sentences. Single-sentence paragraphs used as deliberate weight drops — *"Ini tahun ketiga."* is the clearest example: three words, enormous mass. Action sequences are numbered and clipped ("Tebasan pertama... Kedua... Ketiga..."), almost like a combat log. Exposition sentences are longer and journalist-adjacent — structured, not florid.

## Dialogue Style
Minimal but purposeful. All-caps guild messages do heavy lifting for community voice and scene punctuation. Silence is itself a dialogue choice. Economy should be watched as actual conversations grow longer.

## Pacing Tendencies
Opens in medias res → pulls back for context → returns to present. Not afraid to stop action cold for a backstory block. The risk is exposition density, but tone consistency carries it. The flashback reads like the narrator's dry footnote, not an interruption.

## Strengths
- Economy — nothing is decorative, every sentence earns its place
- Combat sequences: clean, satisfying, not flashy
- World-building through behavior and context, not info-dumps
- Characterization purely through action — no adjective-dumping
- Strong closing beats (action as inaction: Bima's fingers hovering)
- The dual-camera (in-game Bentala / real-world Bima) handled seamlessly

## Watch Out For
- Exposition blocks can run dense — as scenes grow, watch that "narrator explains things" mode doesn't crowd out dramatized moments
- Action lines meant to feel automatic should have syntax that mirrors effortlessness — avoid knotty constructions mid-combat
- Typo pattern: watch phonetic misspellings under speed (e.g. *simestris* → *simetris*)
