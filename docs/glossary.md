# Glossary

| Istilah | Penjelasan |
|---------|-----------|
| **Aggro** | Monster menargetkan player |
| **DPS** | Damage Per Second, role attacker |
| **Tank** | Role penahan damage, protect party |
| **Healer** | Role penyembuh |
| **Loot** | Item drop dari monster |
| **Spawn** | Monster muncul di lokasi |
| **Grind** | Farming monster berulang untuk exp/loot |
| **Party** | Grup player hunting bersama |
| **Guild** | Komunitas besar, war buff |
| **Enhance** | Upgrade gear (+1 to +15) |
| **Reskin** | Monster model sama, warna beda |
| **Meta** | Strategi paling optimal |
| **Solo** | Hunting sendiri |
| **Mount** | Kendaraan, bisa terbang |
| **Title** | Penghargaan, stat boost |
| **Buff** | Temporary stat boost |
| **Debuff** | Negative effect |
| **Stun** | Monster tidak bisa gerak |
| **Blind** | Monster tidak bisa lihat |
| **Fall Damage** | Damage dari jatuh |
